# Ender Evac 服务器全自动跑路工具

[GitHub Repository](https://github.com/07-iceven/EnderEvac)

这是 2026 年愚人节项目 —— **Ender Evac**。

## 项目简介
Ender Evac 是一款用于 Minecraft 服务器“全自动跑路”的自动化工具。它能有效解决服务器长期无人在线时的资源浪费问题，并为玩家留下最后一份“开源大礼包”。

## 核心功能
- **不活跃监测**：自动检测服务器内玩家的在线状态及空置时间。
- **跑路协议自动化**：当不活跃时间达到设定阈值时，程序将自动执行以下操作：
    - **服务器关停**：优雅地关闭 Minecraft 服务进程。
    - **开源归档**：将服务器核心文件自动推送到 GitHub 存储库，实现真正的“全图开源”。
    - **喜报推送**：向 QQ 群、官网等预设渠道自动发送跑路公告及开源地址。
    - **关机节能**：执行最终关机操作，节省电费。

## 现状说明
**重要提示：由于开发者技术水平限制，目前该项目尚未开发实际的后端执行功能。**
当前的 Web 界面仅为一个**功能概念和 UI 原型演示**。所有跑路流程目前均为前端模拟展示。
